# Sinhala Font Files

Place the following font files in this directory for Sinhala PDF certificate generation:

## Required files:
1. `NotoSerifSinhala-Regular.ttf`
2. `NotoSerifSinhala-Bold.ttf`

## Download:
These fonts are available free from Google Fonts:
https://fonts.google.com/noto/specimen/Noto+Serif+Sinhala

Direct download:
https://github.com/google/fonts/tree/main/ofl/notoserifsinhala

## Without these files:
The system will fall back to English (Helvetica) for certificate generation.
All other functionality works normally.

## Installation:
npm run download-fonts
  OR manually copy the .ttf files to this directory.
